# LLM Wiki Agent 스키마 (AGENTS.md)

Karpathy LLM Wiki 패턴 기반 (2026년 4월)
SwikiSwiki (Comanche/Squeak) + Ollama gemma4:latest

## 위키 구조

이 위키는 SwikiSwiki XML 페이지 컬렉션에 저장됩니다.
페이지 300번부터 LLM Wiki 전용으로 사용합니다.

- **300번**: LLM Wiki 인덱스 (전체 목차)
- **301번**: LLM Wiki 로그 (작업 기록)
- **302번~**: 엔티티/컨셉 페이지 자동 생성

## 페이지 유형

1. **엔티티 페이지**: 사람, 조직, 시스템, 제품 등 구체적 개체
2. **컨셉 페이지**: 기술, 방법론, 아이디어 등 추상적 개념

## SwikiSwiki 텍스트 형식 규칙

- 목록 항목: `- 내용`
- 하위 목록: `-- 내용`
- 위키 링크: `*페이지이름*` 또는 `*페이지번호*`
- **마크다운 헤딩(#) 사용 금지** — SwikiSwiki가 지원하지 않음
- 섹션 구분은 빈 줄과 일반 텍스트로

## 수집(Ingest) 워크플로우

1. 새 소스 문서 읽기
2. 핵심 엔티티와 컨셉 추출
3. 기존 위키 인덱스 확인 (중복 방지)
4. 새 페이지 생성 또는 기존 페이지 업데이트
5. 인덱스 및 로그 갱신

## 품질 기준

- 각 페이지는 하나의 엔티티/컨셉에 집중
- 내용은 200~500자, 간결하게
- 관련 페이지 상호 링크 필수
- 한국어 우선, 기술 용어는 영어 병기 허용
- 출처(소스 이름) 명시
